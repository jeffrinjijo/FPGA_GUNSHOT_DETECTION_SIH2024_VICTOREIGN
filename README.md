# Gunshot Detection and Localization System
### Smart India Hackathon (SIH) 2024 Finalist

## Overview

The Gunshot Detection and Localization System is an intelligent surveillance solution developed for Smart India Hackathon (SIH) 2024. The project aims to improve public safety by automatically detecting gunshots, identifying the firearm type, and estimating the direction of the sound source using multiple microphones.

Unlike traditional surveillance systems that depend solely on CCTV footage, this system uses audio signal processing techniques to detect gunfire even in low visibility or obstructed environments.

---

## Problem Statement

Gun-related incidents require immediate response from law enforcement agencies. Traditional surveillance systems rely primarily on visual monitoring, which may fail during poor lighting conditions, long distances, or obstructed views.

This project addresses these limitations by analyzing acoustic signals generated by gunfire. Using a microphone array, digital signal processing, and machine learning, the system can detect gunshots, classify weapon types, and estimate the direction from which the sound originated.

---

## Objectives

- Detect gunshot sounds in real time.
- Differentiate gunshots from environmental noise.
- Classify different firearm categories.

## Dataset

The model was trained using publicly available gunshot audio datasets containing recordings from multiple firearm categories.

### Firearm Categories

- AK-47
- AK-12
- IMI Desert Eagle
- M16
- M249
- MG-42
- MP5
- Zastava M92

The dataset includes hundreds of audio recordings captured under different environmental conditions to improve model generalization.

---

## Audio Preprocessing

Each audio sample undergoes several preprocessing steps before training.

- Audio loading
- Resampling
- Silence removal
- Band-pass filtering
- Normalization
- Noise reduction
- Windowing

These preprocessing techniques improve the quality of extracted features and increase model performance.

---

## Feature Extraction

Mel Frequency Cepstral Coefficients (MFCCs) were extracted from each audio recording.

Additional extracted features include:

- Spectral Centroid
- Zero Crossing Rate
- Chroma Features
- RMS Energy
- Spectral Roll-off

These features capture the acoustic characteristics of different firearms.

---

## Gunshot Localization

To estimate the location of the gunshot, the system uses multiple microphones.

### Localization Method

- Time Difference of Arrival (TDOA)
- Cross Correlation
- Direction of Arrival (DOA) Estimation

By measuring the arrival time differences between microphones, the system estimates the direction from which the gunshot originated.

## Workflow

1. Capture audio using a microphone array.
2. Preprocess the recorded audio.
3. Extract MFCC and spectral features.
4. Classify firearm type using CNN.
5. Compute Time Difference of Arrival (TDOA).
6. Estimate the direction of the gunshot.
7. Generate an alert for security personnel.
